Media Queries

What is a Media Query?

Media queries is a CSS3 module allowing content rendering to adapt to conditions such as screen resolution (e.g. smartphone screen vs. computer screen). It became a W3C recommended standard in June 2012, and is a cornerstone technology of responsive web design

Media Queries should be organic, and based on the design of the website, or a specific device.

What does a Media Query look like?( There are 2 parts of a Media Query. )

Part 1:

Define the name of the Media Query.You can give your Media Query a name inside the _mixins.scss file.

// Small devices

@mixin sm {

@media (min-width: #{$screen-sm-min}) {

@content;

}

}

My template folder gave you 4 Media Queries with the names of:

  • sm ( small )
  • md ( medium )
  • lg ( large )
  • xl ( extra large )

Part 2:

Define the size of the Media Query.You will set the size of the Media Query inside the _variables.scss file.

// Small tablets and large smartphones (landscape view)

$screen-sm-min: 576px;

My template folder gave you 4 Media Queries sizes of:

  • 567px ( small )
  • 768px ( medium )
  • 992px ( large )
  • 1200px ( extra large )
